Machine Learning Projects

This repository contains two machine learning projects implemented using Python and Scikit-learn.
 
Projects

 1. Diabetes Prediction

* Predicts the likelihood of diabetes using medical data
* Includes data preprocessing, model training, and evaluation
* Tech: Python, Pandas, NumPy, Scikit-learn

2. Movie Recommendation System

* Recommends movies using content-based filtering
* Calculates similarity between movies to generate suggestions
* Tech: Python, Pandas, NumPy, Scikit-learn
